CMP Implementation Documentation

 If you need to start building your web application on behalf of Gentics CMP products, this is the right place to start from.

Versions

Before proceed, please make sure that you are reading the documentation for the correct Gentics CMP Product version, that corresponds to your project. Currently, the implementation documentation exists for the following product versions of Gentics CMP (each is clickable):

  • 8.2
  • 8.1
  • 8.0

The available and supported versions can be found here. If you are starting you project from the very beginning, it is reasonable to pick the latest supported Gentics CMP Product version. The version of the current documentation page can be checked at the top right corner of this page, or by looking for the content of <button class='current-version'>, if you are using a HTML page parser.
