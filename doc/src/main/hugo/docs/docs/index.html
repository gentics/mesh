Gentics Mesh is a headless or API-first CMS shipped with a single JAR file.
This is the right place for getting started using Gentics Mesh.

  • The TL;DR section will briefly explain everything you should know and what makes Gentics Mesh special.
  • Our Quickstart guide will let you play with Gentics Mesh in seconds - No installation is required. Just download a single JAR file!
  • You will find an overview of the core concepts, a complete list of all features and references for available APIs, such as REST and GraphQL.
  • To get you started, we have collected a set of samples for different platforms, i.e. JavaScript, Java, GO, and PHP .
  • The administration guide, will provide information on how to configure and run Gentics Mesh in your environment.
